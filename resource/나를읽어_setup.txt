1. MySQL Setup
  1.1 MySQL 설치시 Root 계정 설정 id/password: root/root
  1.2 Database 설치 및 Root에 접근 권한 설정

	create database simpleweb;
	use simpleweb;
	grant all privileges on simpleweb.* to root@localhost ;
	flush privileges;

  1.3 Database table 설정
    databasescript.sql 파일을 MySQL Workbench에서 실행시킨다.
